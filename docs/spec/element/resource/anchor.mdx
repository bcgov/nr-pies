Represents an anchor. An anchor serves as a foundational reference point within the PIES system.

## Scope

This resource type may be transmitted within a Record message by setting `record_kind` to `ANCHOR`. However, Anchor is defined as an independent, composable resource type and may be used in other message contexts.

Refer to [Record](/docs/spec/element/message/record) for details on the message envelope pattern and how resource types are selected.

## Structure

This structure contains no properties and is functionally equivalent to an empty object.

ignore any unexpected properties that may be present.
